Requirements :
Python3, pip3 
pandas, pickle (pip3 install pickle-mixin)
numpy, scikit-learn
matplotlib, seaborn

1. To check existing file implementation :

Run python3 classifier.py

*Note-all files are already trained and stored in folder.



2. To remove files and re-train model again and then check implementation :
Run make clean
Run make

*Note- Random Forest takes ~20 minutes to train.
